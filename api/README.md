# Urban Sport Store Backend

This backend provides a secure REST API for the Urban Sport Store e-commerce platform.

## Features
- Authentication and user profiles
- Catalog, categories, brands, products and variants
- Cart, coupons and checkout flow
- Orders, simulated payments, inventory and admin endpoints
- Swagger docs at /api/docs

## Scripts
- npm install
- npm run db:generate
- npm run db:migrate
- npm run db:seed
- npm run dev
- npm run test
- npm run build

## Local URLs
- Health: http://localhost:4000/health
- Swagger: http://localhost:4000/api/docs
